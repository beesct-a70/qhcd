# Hướng dẫn thêm ảnh vào slider

Để thêm ảnh thật cho các khóa học trước đây, hãy làm theo các bước:

1. Đặt 6 file ảnh vào thư mục `/public/` với tên:
   - qhcd1.png (Khóa học K22)
   - qhcd2.png (Khóa học K21)
   - qhcd3.png (Khóa học K20)
   - qhcd4.png (Khóa học K19)
   - qhcd5.png (Khóa học K18)
   - qhcd6.png (Khóa học K17)

2. Kích thước ảnh đề xuất: 1200x800 pixels (tỷ lệ 3:2)

3. Định dạng ảnh: PNG, JPG, hoặc WebP

4. Sau khi thêm ảnh, component ImageSlider sẽ tự động hiển thị ảnh thật thay vì placeholder
